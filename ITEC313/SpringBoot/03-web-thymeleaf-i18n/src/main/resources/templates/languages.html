How Internationalization Works

This application uses Spring Boot's internationalization features:

  • Message Sources: Properties files containing translated text
  • Locale Resolver: Determines the current locale from session or request
  • Locale Change Interceptor: Handles locale changes via URL parameters
  • Thymeleaf Integration: Automatic message resolution in templates
Available Message Files:
  • messages.properties - Default (English)
  • messages_es.properties - Spanish
  • messages_fr.properties - French (to be added)
  • messages_de.properties - German (to be added)
  • messages_it.properties - Italian (to be added)
