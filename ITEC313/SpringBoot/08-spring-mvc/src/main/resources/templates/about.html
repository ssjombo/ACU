About Us

This is a Spring MVC application demonstrating various features.

Spring MVC Architecture

This application demonstrates the classic Spring MVC architecture with the following components:

DispatcherServlet

The front controller that handles all requests and routes them to appropriate handlers.

Controllers

Handle HTTP requests and return view names or data to be rendered.

ViewResolver

Resolves view names to actual view implementations (Thymeleaf templates).

Model

Carries data from controllers to views for rendering.

Key Features Demonstrated

  • Form Handling: Contact form with validation using Jakarta Validation
  • File Upload: MultipartFile handling with file storage
  • Error Handling: Global exception handling with @ControllerAdvice
  • Interceptors: Request/response logging with HandlerInterceptor
  • Static Resources: CSS, JS, and image serving
  • Templating: Thymeleaf templates with dynamic content
